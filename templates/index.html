Usage

Example POST request to the register route 

            POST https://naijaemoji.herokuapp.com/register
            Request body:
            {
                'username' : 'johndoe',
                'password' : 'password'
            }
The response the to the following request includes the message, the response code and the some other data. On succesful registration, the following json reponse will be returned 
            
                Response body:
                {
                'username'  : 'johndoe',
                'password'  : 'someEncryptedString',
                'message'   : 'Registration succesful'
                }
